# Changelog

This changelog only contains the changes that are unreleased. For changes for individual releases, please visit the
[releases](https://github.com/ATLauncher/ATLauncher/releases) page on GitHub.

## 3.4.41.0

### New Features

- Allow launching servers from the cli with "--launch-server name" and optionally "--server-gui" to show the gui thanks to @ASchoe311
- Support installing datapacks to the datapacks folder and support for adding and managing them
- Add Modrinth download metadata to Modrinth CDN downloads [#1028]
- Add Catppuccin themes thanks to @tildeman

### Fixes

- Fixed slow and jumpy scrolling in the Edit Mods dialog on macOS thanks to @nllvoid [#1032]
- Add api key on CurseForge cdn download links as required by CurseForge
